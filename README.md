# CatchPokemon

CatchPokemon is an app that allows you to catch pokemons

## Installation

Install [.NET core SDK](https://dotnet.microsoft.com/download)

## Usage

```bash
cd src/Catch_Pokemons
dotnet build
dotnet run
//or use
dotnet run bild --project src/Catch_Pokemons
```
